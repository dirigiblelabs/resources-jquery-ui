# jQuery UI for Eclipse Dirigible

jQuery UI Resource Plugin for the [Eclipse Dirigible](https://github.com/eclipse/dirigible) v3.x and above.

Build the [Eclipse Dirigible](https://github.com/eclipse/dirigible), deploy on e.g. Tomcat Web Container and go to location:

> http://localhost:8080/services/v3/web/resources/jquery-ui/{version}/jquery-ui.min.js
